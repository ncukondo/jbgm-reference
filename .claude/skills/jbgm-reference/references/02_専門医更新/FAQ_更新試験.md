# FAQ: 更新試験

- 出典: https://jbgm.org/qa/

## 受験について

### Q1.＜NEW＞IBTは「e-テキストの利用可」とのことですが、公式のe-テキスト等はあるのでしょうか。どこからダウンロード(購入)すれば良いのでしょうか。

A1.e-テキストとは、一般的に使用されるテキストや受験者が事前に作成したテキスト等のことであり、当委員会が公式に提供するものではありません。

- [参照元](https://jbgm.org/qa/%ef%bc%9cnew%ef%bc%9eibt%e3%81%af%e3%80%8ce-%e3%83%86%e3%82%ad%e3%82%b9%e3%83%88%e3%81%ae%e5%88%a9%e7%94%a8%e5%8f%af%e3%80%8d%e3%81%a8%e3%81%ae%e3%81%93%e3%81%a8%e3%81%a7%e3%81%99%e3%81%8c%e3%80%81)

### Q2.＜NEW＞IBT試験について、自宅や職場など第三者がいない環境でオンライン受験が可能、電話やチャット、写真撮影、スクリーンショットは禁止との規定ですが、不正行為の防止と対応はどのように行われる予定でしょうか。

A2.既に米国で実施されているIBTシステムにより、AIおよび試験担当者が不正チェックを行います。不正とみなされた場合は試験が強制終了される可能性があります。

- [参照元](https://jbgm.org/qa/%ef%bc%9cnew%ef%bc%9eibt%e8%a9%a6%e9%a8%93%e3%81%ab%e3%81%a4%e3%81%84%e3%81%a6%e3%80%81%e8%87%aa%e5%ae%85%e3%82%84%e8%81%b7%e5%a0%b4%e3%81%aa%e3%81%a9%e7%ac%ac%e4%b8%89%e8%80%85%e3%81%8c%e3%81%84)

### Q3.更新（IBT）試験はいつ受験すれば良いですか？

A3.更新試験は総合診療専門医認定後の4年目および5年目の最大2回受験することができます。また、この2回において未受験または不合格の場合は猶予申請を認め、承認された場合は更新年（専門医認定後6年目）に1回のみ受験が認められます。詳しくは以下リンクよりご確認ください。・『総合診療専門医の認定・更新』別添資料1・総合診療専門医更新試験情報（概要および実施要綱）
