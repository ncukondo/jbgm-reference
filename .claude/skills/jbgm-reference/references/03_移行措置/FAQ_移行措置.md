# FAQ: 移行措置

- 出典: https://jbgm.org/qa/

## 制度/専門医試験

### Q1.特任指導医の移行措置による専門医試験受験はいつまで認められますか。

A1.移行措置による特任指導医の専門医試験受験の期限は2030年までで、それまでは何度でも受験することができます。詳しくは以下をご確認ください専門医への移行措置に係る規則について＜2024年1月＞

- [参照元](https://jbgm.org/qa/%e8%aa%8d%e5%ae%9a%e6%9c%9f%e9%96%93%e3%81%8c2023%e5%b9%b43%e6%9c%8831%e6%97%a5%e3%81%be%e3%81%a7%e3%81%ae%e7%89%b9%e4%bb%bb%e6%8c%87%e5%b0%8e%e5%8c%bb%e3%81%8c%e3%80%81%e7%a7%bb%e8%a1%8c%e6%8e%aa)
    - [専門医への移行措置に係る規則について＜2024年1月＞](https://jmsb.box.com/s/vq97588pssi046xo5wvnr3srr1811b7a)

### Q2.特任指導医は全員、総合診療専門医の移行措置に従って総合診療専門医試験を受験しなければならないのでしょうか。

A2.全ての特任指導医が総合診療専門医試験を受験しなければならない規定はありませんが、専門研修の指導医は原則として専門医であることが求められており、いずれかの時点で特任指導医制度が廃止される方向であることを考慮していただく必要があります。特任指導医から専門医への移行措置の期限である2030年度までの専門医試験の受験を前向きにご検討いただきますようお願いします。

- [参照元](https://jbgm.org/qa/%e7%89%b9%e4%bb%bb%e6%8c%87%e5%b0%8e%e5%8c%bb%e3%81%af%e5%85%a8%e5%93%a1%e3%80%81%e7%b7%8f%e5%90%88%e8%a8%ba%e7%99%82%e5%b0%82%e9%96%80%e5%8c%bb%e3%81%ae%e7%a7%bb%e8%a1%8c%e6%8e%aa%e7%bd%ae%e3%81%ab)

### Q3.専門医認定試験に合格すれば、翌年度以降は指導医として活動できるのでしょうか。

A3.移行措置により専門医試験を受験して合格、専門医に認定された特任指導医は、その他の認定要件も満たせば指導医になることができます。（正式な）指導医の認定要件の詳細はプログラム整備基準をご確認ください。

- [参照元](https://jbgm.org/qa/2022%e5%b9%b4%e5%ba%a6%e3%81%ab%e5%8f%97%e9%a8%93%e3%81%97%e3%81%a6%e5%90%88%e6%a0%bc%e3%81%99%e3%82%8c%e3%81%b0%e3%80%812023%e5%b9%b4%e5%ba%a6%e4%bb%a5%e9%99%8d%e3%81%ab%e6%8c%87%e5%b0%8e%e5%8c%bb)

## 対象者

### Q1.【家庭医療専門医からの移行における医療資源の乏しい地域の研修について】医療資源の乏しい地域での研修の証明はどうすればよいか？

A1.移行措置対象者が研修を修了した家庭医療専門研修プログラムのプログラム責任者が発行する証明書（機構の指定する様式）を提出してください。猶予を希望される場合は、猶予申請書（機構の指定する様式）の提出をお願いします。指定様式は、「平成29年度総合診療専門医の研修の暫定措置について」にございます。

- [参照元](https://jbgm.org/qa/%ef%bc%9cnew%ef%bc%9e%e3%80%90%e5%ae%b6%e5%ba%ad%e5%8c%bb%e7%99%82%e5%b0%82%e9%96%80%e5%8c%bb%e3%81%8b%e3%82%89%e3%81%ae%e7%a7%bb%e8%a1%8c%e3%81%ab%e3%81%8a%e3%81%91%e3%82%8b%e5%8c%bb%e7%99%82-2)

### Q2.【家庭医療専門医からの移行における医療資源の乏しい地域の研修について】医師臨床研修（初期研修）中の研修実績も含めて良いのか？

A2.総合診療専門研修では、総合診療専門研修中または研修期間修了後の期間のみ認めています。同様に、移行措置対象者においては、家庭医療専門研修中および改定医療専門研修修了後の勤務についてのみお認めします（医師臨床研修中や他領域での研修中はお認めしません）。

- [参照元](https://jbgm.org/qa/%ef%bc%9cnew%ef%bc%9e%e3%80%90%e5%ae%b6%e5%ba%ad%e5%8c%bb%e7%99%82%e5%b0%82%e9%96%80%e5%8c%bb%e3%81%8b%e3%82%89%e3%81%ae%e7%a7%bb%e8%a1%8c%e3%81%ab%e3%81%8a%e3%81%91%e3%82%8b%e5%8c%bb%e7%99%82)

### Q3.【家庭医療専門医からの移行における医療資源の乏しい地域の研修について】常勤でなければならないなどの雇用形態の制限はあるのか。

A3.ブロックでの研修、週1日などでの単位制での勤務、あるいは両者の組合せで、必要期間（6か月）またはそれに相当する単位数を取得していれば、雇用形態は問いません。

- [参照元](https://jbgm.org/qa/%e3%80%90%e5%ae%b6%e5%ba%ad%e5%8c%bb%e7%99%82%e5%b0%82%e9%96%80%e5%8c%bb%e3%81%8b%e3%82%89%e3%81%ae%e7%a7%bb%e8%a1%8c%e3%81%ab%e3%81%8a%e3%81%91%e3%82%8b%e5%8c%bb%e7%99%82%e8%b3%87%e6%ba%90%e3%81%ae-2)

### Q4.【家庭医療専門医からの移行における医療資源の乏しい地域の研修について】研修地は、総合診療専門研修プログラムの研修施設に限定されるのか？

A4.総合診療専門研修プログラム整備基準に記載されている「医療資源の乏しい地域」に所在する施設での研修歴が必要です。ただし、移行措置対象者が2017年度に家庭医療専門研修を開始した時点では、総合診療専門医制度は始まっていなかったことから、研修施設が総合診療専門研修プログラムの基幹施設または連携施設になっている必要はありません。

- [参照元](https://jbgm.org/qa/%e3%80%90%e5%ae%b6%e5%ba%ad%e5%8c%bb%e7%99%82%e5%b0%82%e9%96%80%e5%8c%bb%e3%81%8b%e3%82%89%e3%81%ae%e7%a7%bb%e8%a1%8c%e3%81%ab%e3%81%8a%e3%81%91%e3%82%8b%e5%8c%bb%e7%99%82%e8%b3%87%e6%ba%90%e3%81%ae)

### Q5.内科領域・小児科領域・救急領域・その他の領域の指導医として6ヶ月以上専攻医を指導している場合、移行措置による専門医試験の受験は可能ですか。

A5.移行措置の対象となる特任指導医は、総合診療専門研修Ⅰ・Ⅱの特任指導医のみとなります。

- [参照元](https://jbgm.org/qa/%e5%86%85%e7%a7%91%e9%a0%98%e5%9f%9f%e3%83%bb%e5%b0%8f%e5%85%90%e7%a7%91%e9%a0%98%e5%9f%9f%e3%83%bb%e6%95%91%e6%80%a5%e9%a0%98%e5%9f%9f%e3%83%bb%e3%81%9d%e3%81%ae%e4%bb%96%e3%81%ae%e9%a0%98%e5%9f%9f)

### Q6.第3条（２）「（2004年以降に医師免許を取得した者は2017年3月31日までに医師臨床研修を修了している）特任指導医」について、2003年以前に医師免許を取得したため医師臨床研修を行っていませんが、移行措置の対象にはならないのでしょうか？

A6.2017年3月31日までに医師臨床研修修了が必要となるのは、新医師臨床研修制度の創設に伴い臨床研修が必修化された2004年以降に医師免許を取得した特任指導医であり、2003年以前に医師免許を取得した特任指導医は、実地修練や臨床研修の有無に関わらず移行措置の対象となります。

- [参照元](https://jbgm.org/qa/%ef%bc%9cnew%ef%bc%9e%e7%ac%ac3%e6%9d%a1%ef%bc%88%ef%bc%92%ef%bc%89%e3%80%8c%ef%bc%882004%e5%b9%b4%e4%bb%a5%e9%99%8d%e3%81%ab%e5%8c%bb%e5%b8%ab%e5%85%8d%e8%a8%b1%e3%82%92%e5%8f%96%e5%be%97%e3%81%97)

### Q7.第3条「他領域の新専門医制度のプログラムで研修を行う者は、移行措置の対象外とする」について、既に他領域の専門医研修を修了（専門医を取得）している場合は対象になりますか？

A7.2018年度から開始された新専門医制度で専門研修中の方、または専門研修を修了した方は、移行措置の対象外です。また専門研修の有無に関わらず、2016年度（2017年3月31日）以前に臨床研修を修了していなければ、移行措置の対象外となります。

- [参照元](https://jbgm.org/qa/%ef%bc%9cnew%ef%bc%9e%e7%ac%ac3%e6%9d%a1%e3%80%8c%e4%bb%96%e9%a0%98%e5%9f%9f%e3%81%ae%e6%96%b0%e5%b0%82%e9%96%80%e5%8c%bb%e5%88%b6%e5%ba%a6%e3%81%ae%e3%83%97%e3%83%ad%e3%82%b0%e3%83%a9%e3%83%a0)
