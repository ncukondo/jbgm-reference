期間短縮を伴うダブルボード対象者 
オンライン研修手帳J-GOAL 入力に関する注意点 
 
＊期間短縮を伴うダブルボード対象者 
1.内科→総合診療 
(1)通常型（1 年短縮） 
a.総合診療専門研修中の総合診療専門研修ⅠおよびⅡ、小児科研修、救急科研修につい
て、オンライン研修手帳に記録してください。 
b.「⑥ 指導医との定期的な振り返りミーティングの記録について 
・「2 年次終了時の振り返り」「3 年次終了時の振り返り」「総合診療専門研修終了時の
振り返り」に入力してください。「指導医の形成的評価」も必要です。 
・「1 年次終了時の振り返り」は入力の必要はありません。 
 
(2)特例型（2 年短縮） 
a. 総合診療専門研修中の総合診療専門研修Ⅰ、小児科研修、救急科研修に加え、内科専門
研修中の総合診療専門研修Ⅱに相当する期間の研修について、オンライン研修手帳に記録
してください。内科専門研修中の総合診療専門研修Ⅱに相当する期間の研修については、
当該期間中にオンライン研修手帳に入力し、かつ、デュアル指導医（内科専門研修プログ
ラムの指導医と日本専門医機構総合診療(特任)指導医の両方の資格を有する指導医）の評
価を受ける必要があります。 
b.「⑥ 指導医との定期的な振り返りミーティングの記録」について 
・「2 年次終了時の振り返り」は、内科専門研修中の総合診療専門研修Ⅱに相当する期間
の研修について入力してください。「指導医の形成的評価」も必要です。 
・「3 年次終了時の振り返り」は、総合診療専門研修における研修について入力してくだ
さい。指導医の形成的評価」も必要です。 
・「総合診療専門研修終了時の振り返り」は、内科専門研修中の総合診療専門研修Ⅱに相
当する期間の研修および総合診療専門研修における研修について入力してください。「指
導医の形成的評価」も必要です。 
・「1 年次終了時の振り返り」は入力の必要はありません。 
 
2.救急科→総合診療 
a. 総合診療専門研修中の総合診療専門研修ⅠおよびⅡ、小児科研修について、オンライン
研修手帳に記録してください。 
・ただし、2024 年度までに総合診療専門研修を開始した場合には、救急科専門研修中に
行った小児科研修における研修実績は、総合診療専門研修開始時にオンライン研修手帳 J-
GOAL に自己評価を入力してください。自己評価が、下記に示す各項目の到達度を達成し

ている必要があります。自己評価で下記に示す各項目の到達度を達成できない場合は、未
達成の部分について総合診療専門研修Ⅰの期間に総合診療指導医または総合診療特任指導
医による指導医評価を受けることが必要となります。指導医評価と上記の自己評価をあわ
せて、下記に示す到達度を達成してください。 
 
【救急科専門研修中の小児科研修の自己評価】 
Ⅷ. 小児救急 一般目標・診療能力 
（１）小児救急 診療能力 
「意識、呼吸、脈拍、CRT（毛細血管充満時間）などから全身状態を迅速・適切に評価
し、緊急度をトリアージできる」「特別な医療機器を用いない一次救命処置、救急薬剤、
機器を用いての二次救命処置ができる」のいずれも自己評価が A または B 以上。いずれ
かが C または N であれば不可。ただし、以下の代替措置がある。 
・小児 2 次救命処置コースである PALS や JPLS、あるいは小児蘇生のトレーニングが
含まれている AHA BLS コース、または AHA PEARS コースの修了を確認できる書類が
提出されていることで代えることも可とする。 
（２）小児救急 一般目標および診療能力 12 項目中、8 項目以上の自己評価が A または 
B であること。 
 
Ⅷ. 小児総合診療 一般目標・診療能力 
80%以上の項目の自己評価が B 以上であること。ただし、自己評価が N である項目を分
母に含める。 
 
Ⅸ．手技、症候・疾患 
80%以上の項目の自己評価が B 以上であること。 
 
Ⅹ．疾患 
①知識として十分に理解すべき代表的疾患 
②治療もしくは初期対応まで修得すべき代表的疾患 
①②いずれも自己評価 B 以上が 80%以上であること。
