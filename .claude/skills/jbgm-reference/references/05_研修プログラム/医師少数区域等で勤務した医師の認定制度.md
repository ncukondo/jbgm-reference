# 医師少数区域等で勤務した医師の認定制度

- 出典: https://jmsb.app.box.com/s/sbls6zobsy7kg6pxa1vkbginlwseliat
- 収集元ページ: https://jbgm.org/menu/基準・細則等/

【重要なお知らせ】「医師少数区域等で勤務した医師の認定制度」による認定の申請について 
 
2018 年に医療法及び医師法の⼀部を改正する法律が公布され、厚⽣労働⼤⾂による医師の認定に関する
事項等について、2020 年4 ⽉1 ⽇で施⾏されました。この改正により、「医師の確保を特に図るべき区域
（医師少数区域等）における医療提供の経験を有する医師」を厚⽣労働⼤⾂が認定することになりました
が、この認定を受けることにより、 
 
１）「医師少数区域経験認定医師」である旨が医籍に登録される 
２）「医師少数区域経験認定医師」として広告が可能 
３）2020 年4 ⽉以降に臨床研修を開始した医師については、地域医療⽀援病院の管理者要件となる 
４）医師少数区域等で開業する際に優遇融資あり 
５）認定医師が医師少数区域等で勤務を継続するためのスキルアップを⽬的とした研修費等について、医
療機関に対して国の補助⾦が⽀給される 
 
などのメリットがあります。認定の対象となるのは、以下の１）または２）を満たし、かつ、その区域に
おいて３）の業務を⾏った場合とされています。 
 
１）医師少数区域等（下記の厚⽣労働省ウェブサイトに⼀覧表あり）に所在する医療機関で、2020 年4
⽉1 ⽇以降に連続6 か⽉以上勤務（臨床研修期間は含めない） 
２）医師免許を取得後9 年以上経過している場合（＝10 年⽬以降）は、断続的な勤務⽇数の合計が180 
⽇以上でも可 
３）該当する地域において以下の全ての業務を⾏っていること 
① 個々の患者の⽣活状況を考慮し、幅広い病態に対応する継続的な診療や保健指導 
② 他の医療機関や、介護・福祉事業者等との連携 
③ 地域住⺠に対する健康診査や保健指導等の地域保健活動 
 
なお、申請⽅法を含めたこの制度の詳細については、厚⽣労働省ウェブサイト「医師確保対策」ページの
「医師少数区域等で勤務した医師の認定制度」をご確認下さい。厚生労働省の指定する「医師少数区域等」
 は、総合診療専門研修プログラム整備基準で既定 している医療資源の乏しい地域の一部であり、医療資源
の乏しい地域であっても「医師少数区域等」に該当しない地域もあります。 
詳細は各都道府県窓口にお問い合わせください。 
https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/kenkou_iryou/iryou/kinkyu/index.html
 
 総合診療専⾨医検討委員会が運営する総合診療専⾨研修制度では、「医療資源の乏しい地域」での6 か⽉
以上の勤務を必須としており、医師少数区域等も「医療資源の乏しい地域」として認定されています。ま
た、認定の対象となる上記3 種類の業務はいずれも、総合診療専⾨研修において経験すべき業務に含まれ
ていますので、この制度はまさしく総合診療専攻医・専⾨医のための制度であるとも⾔えます。 
そこで、既に「医療資源の乏しい地域」で6 か⽉以上の研修をされた⽅は、ご⾃⾝がこの認定を受ける資
格をお持ちである可能性がありますので、確認されることをお勧めいたします。また、これから「医療資
源の乏しい地域」での研修を⾏う⽅におかれましては、上記制度についてご参考になさって下さい。
