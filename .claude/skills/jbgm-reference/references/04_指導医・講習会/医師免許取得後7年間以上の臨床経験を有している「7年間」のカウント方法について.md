# 医師免許取得後7年間以上の臨床経験を有している「7年間」のカウント方法について

- 出典: https://jmsb.app.box.com/s/rwzn2fpohlnafdfndwfm3ro4s7et0elx
- 収集元ページ: https://jbgm.org/menu/認定・更新/

---

特任指導医の認定要件、医師免許取得後7年間以上の臨床
経験を有している「7年間」のカウント⽅法について
医籍登録した年度を起点として医師免許取得後臨床経験年数をカウントします(臨床研修指導医資格と同じカウントの⽅法です）。
例えば、医籍登録⽇が2017年8⽉18⽇の場合、2017年4⽉1⽇が卒後臨床研修開始⽇、2019年3⽉31⽇の卒後臨床研修修了⽇をもって、所定
された2年間の臨床研修を修了したことになり、その場合、2024年3⽉31⽇で医籍登録した後7年間経過、2024年4⽉1⽇より卒後8年⽬にな
ります。
従いまして、上記医師の場合は2024年4⽉1⽇より特任指導医の認定が可能となります。
