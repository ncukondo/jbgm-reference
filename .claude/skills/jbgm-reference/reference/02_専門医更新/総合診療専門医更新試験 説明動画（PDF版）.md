受験上の注意事項〜試験問題の取扱い
総合診療専門医更新試験の試験問題の著作権その他一切の知的財産権は、全て日本
専門医機構に帰属しています。
試験問題に関しては、当該受験者本人による試験時間中における試験問題への回答
