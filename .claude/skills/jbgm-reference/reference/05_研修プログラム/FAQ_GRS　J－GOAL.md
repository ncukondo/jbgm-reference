# FAQ: GRS　J－GOAL

- 出典: https://jbgm.org/qa/

## 特任指導医申請

### Q1.J-GOALの手続きができている特任指導医は、指導医番号を記載すれば、医籍登録番号、メールアドレス、生年月日を記載しなくても良いですか。

A1.「指導医番号」はありません。各領域の指導医を一つのシステム（GRS）上でデータ管理を行っていることから、全ての医師が重複なく有する番号（ユニークID）である医籍登録番号を、IDとして用いています。メールアドレスは、連絡手段として用いますので必須です。生年月日は、パスワード紛失時の本人認証のために用います。なお、申請時に記載できなかった指導医の情報は、後日必ずすべての情報を記載して専門医機構事務局に提出してください。

- [参照元](https://jbgm.org/qa/j-goal%e3%81%ae%e6%89%8b%e7%b6%9a%e3%81%8d%e3%81%8c%e3%81%a7%e3%81%8d%e3%81%a6%e3%81%84%e3%82%8b%e7%89%b9%e4%bb%bb%e6%8c%87%e5%b0%8e%e5%8c%bb%e3%81%af%e3%80%81%e6%8c%87%e5%b0%8e%e5%8c%bb%e7%95%aa)

### Q2.名前の横の登録番号欄はどの番号を記載しますか。

A2.医籍登録番号を記入して下さい。

- [参照元](https://jbgm.org/qa/%e5%90%8d%e5%89%8d%e3%81%ae%e6%a8%aa%e3%81%ae%e7%99%bb%e9%8c%b2%e7%95%aa%e5%8f%b7%e6%ac%84%e3%81%af%e3%81%a9%e3%81%ae%e7%95%aa%e5%8f%b7%e3%82%92%e8%a8%98%e8%bc%89%e3%81%97%e3%81%be%e3%81%99%e3%81%8b)

## 研修手帳（経験省察研修録含む）

### Q1.＜NEW＞経験省察研修録について、専門研修中の経験であれば期間やローテート(内科、総合診療Ⅰ・Ⅱ、小児、救急 等)は問わず登録をしてよいでしょうか

A1.総合診療専門研修期間中の経験であれば研修領域は問いませんが、経験省察研修録の指導は、必ず当該プログラムに在籍する総合診療指導医または総合診療特任指導医から受ける必要があります。

- [参照元](https://jbgm.org/qa/%ef%bc%9cnew%ef%bc%9e%e7%b5%8c%e9%a8%93%e7%9c%81%e5%af%9f%e7%a0%94%e4%bf%ae%e9%8c%b2%e3%81%ab%e3%81%a4%e3%81%84%e3%81%a6%e3%80%81%e5%b0%82%e9%96%80%e7%a0%94%e4%bf%ae%e4%b8%ad%e3%81%ae%e7%b5%8c)

### Q2.各項目に対するルーブリックが発表される可能性はありますか。

A2.経験省察研修録は、専攻医が総合診療専門医の7つの資質・能力を修得していく過程で、省察を促し、「示唆に富んだ助言」を提供することを目的としています。また総合診療専門医には、地域によって異なるニーズを的確に把握し、これに応じて自らの修得すべき能力を自ら考え、修得することが求められています。機構が単一のルーブリックを提示することは予定しておりませんので、是非、専攻医が指導医との関わりの中で、自らの目標を見据え、評価する基準を作り上げるその過程を支援していただけるようお願いいたします。

- [参照元](https://jbgm.org/qa/%e5%90%84%e9%a0%85%e7%9b%ae%e3%81%ab%e5%af%be%e3%81%99%e3%82%8b%e3%83%ab%e3%83%bc%e3%83%96%e3%83%aa%e3%83%83%e3%82%af%e3%81%8c%e7%99%ba%e8%a1%a8%e3%81%95%e3%82%8c%e3%82%8b%e5%8f%af%e8%83%bd%e6%80%a7)

### Q3.2018年度専門研修開始の専攻医は、2019年度版研修手帳（暫定印刷版）を使用しても良いですか。

A3.専攻医の希望により経験省察研修録タイプBも含めて使用が可能です。ただし、1人の専攻医が2018年度版と2019年度版を併用することはできず、必ずどちらか一方を選択していただく事になります。

- [参照元](https://jbgm.org/qa/2018%e5%b9%b4%e5%ba%a6%e5%b0%82%e9%96%80%e7%a0%94%e4%bf%ae%e9%96%8b%e5%a7%8b%e3%81%ae%e5%b0%82%e6%94%bb%e5%8c%bb%e3%81%af%e3%80%812019%e5%b9%b4%e5%ba%a6%e7%89%88%e7%a0%94%e4%bf%ae%e6%89%8b%e5%b8%b3)

### Q4.2018年度専攻医の研修手帳（J-GOAL）について

A4.2018年度研修開始の専攻医は、2018年度研修手帳、または、2019年度版研修手帳（暫定印刷版）のいずれかを使用してください。また、2019年度版研修手帳（暫定印刷版）の経験省察研修録にはタイプAとタイプBがありますが、いずれか一方を選択して下さい。なお、年度の異なる研修手帳の混在、タイプの異なる経験省察研修録の混在は、当機構では受理しませんのでご注意下さい。なお、2018年度に研修を開始した3年制以外の専攻医についても、J-GOAL提出を推奨いたします。ただし、従来どおり紙媒体での提出も可とします。

- [参照元](https://jbgm.org/qa/2018%e5%b9%b4%e5%ba%a6%e5%b0%82%e6%94%bb%e5%8c%bb%e3%81%ae%e7%a0%94%e4%bf%ae%e6%89%8b%e5%b8%b3%ef%bc%88j-goal%ef%bc%89%e3%81%ab%e3%81%a4%e3%81%84%e3%81%a6)

### Q5.2019年度専攻医の研修手帳（J-GOAL）について

A5.当初2019年度研修手帳（暫定印刷版）を暫定的に使用していた場合でも、研修手帳のデータをJ-GOALに移行してJ-GOALの方をご使用下さい。なお、経験省察研修録にはタイプAとタイプBがありますが、いずれか一方を選択して下さい。タイプの異なる経験省察研修録の混在は、当機構では受理しませんのでご注意下さい。

- [参照元](https://jbgm.org/qa/2019%e5%b9%b4%e5%ba%a6%e5%b0%82%e6%94%bb%e5%8c%bb%e3%81%ae%e7%a0%94%e4%bf%ae%e6%89%8b%e5%b8%b3%ef%bc%88j-goal%ef%bc%89%e3%81%ab%e3%81%a4%e3%81%84%e3%81%a6)

### Q6.2020年度以降の専攻医の研修手帳（J-GOAL）について

A6.2020年度以降に研修開始の専攻医は、オンライン版研修手帳（J-GOAL）を使用してください。なお、2020年度からは経験省察研修録はタイプBに一本化されました。2019年度研修開始の専攻医のために仕様上はタイプAがアップロードできるようになっていますが、当機構では受理しませんのでご注意下さい。

- [参照元](https://jbgm.org/qa/2020%e5%b9%b4%e5%ba%a6%e4%bb%a5%e9%99%8d%e3%81%ae%e5%b0%82%e6%94%bb%e5%8c%bb%e3%81%ae%e7%a0%94%e4%bf%ae%e6%89%8b%e5%b8%b3%ef%bc%88j-goal%ef%bc%89%e3%81%ab%e3%81%a4%e3%81%84%e3%81%a6)

### Q7.経験省察研修録タイプAは、研修手帳2018年度版では詳細事例20項目＋簡易事例20項目、研修手帳2019年度版ではそれぞれ14項目＋11項目となっていますが、間違いないですか。

A7.間違いありません。2019年度は2018年度版より提出必要数が減りました。

- [参照元](https://jbgm.org/qa/%e7%b5%8c%e9%a8%93%e7%9c%81%e5%af%9f%e7%a0%94%e4%bf%ae%e9%8c%b2%e3%82%bf%e3%82%a4%e3%83%97a%e3%81%af%e3%80%81%e7%a0%94%e4%bf%ae%e6%89%8b%e5%b8%b32018%e5%b9%b4%e5%ba%a6%e7%89%88%e3%81%a7%e3%81%af)

## 実務評価の記録

### Q1.他職種による360度評価を実施するにあたり、URLは評価者一名につき、一つのURLでしょうか（360度評価では他職種の複数名から評価をいただく事になるが、複数名で一つのURLに入力してOKなのか、一名ずつ一つのURLを発行して登録してもらうのか）？同様にmini-CEXについても同じでしょうか。

A1.多職種評価については評価者一名ごと、1回ごとにURLの発行をお願い致します。mini-CEX、CbDについても同様です。

- [参照元](https://jbgm.org/qa/%e4%bb%96%e8%81%b7%e7%a8%ae%e3%81%ab%e3%82%88%e3%82%8b360%e5%ba%a6%e8%a9%95%e4%be%a1%e3%82%92%e5%ae%9f%e6%96%bd%e3%81%99%e3%82%8b%e3%81%ab%e3%81%82%e3%81%9f%e3%82%8a%e3%80%81url%e3%81%af%e8%a9%95)
