【救急科→総合診療】期間短縮を伴うダブルボード対象者 
オンライン研修手帳J-GOAL 入力に関する注意点 
 
a. 総合診療専門研修中の総合診療専門研修ⅠおよびⅡ、小児科研修について、オンライン研修手帳に記
録してください。 
・ただし、2024 年度までに総合診療専門研修を開始した場合には、救急科専門研修中に行った小児科
研修における研修実績は、総合診療専門研修開始時にオンライン研修手帳 J-GOAL に自己評価を入力
してください。自己評価が、下記に示す各項目の到達度を達成している必要があります。自己評価で下
記に示す各項目の到達度を達成できない場合は、未達成の部分について総合診療専門研修Ⅰの期間に総
合診療指導医または総合診療特任指導医による指導医評価を受けることが必要となります。指導医評価
と上記の自己評価をあわせて、下記に示す到達度を達成してください。 
 
【救急科専門研修中の小児科研修の自己評価】 
Ⅷ. 小児救急 一般目標・診療能力 
（１）小児救急 診療能力 
「意識、呼吸、脈拍、CRT（毛細血管充満時間）などから全身状態を迅速・適切に評価し、緊急度をト
リアージできる」「特別な医療機器を用いない一次救命処置、救急薬剤、機器を用いての二次救命処置
ができる」のいずれも自己評価が A または B 以上。いずれかが C または N であれば不可。ただ
し、以下の代替措置がある。 
・小児 2 次救命処置コースである PALS や JPLS、あるいは小児蘇生のトレーニングが含まれている 
AHA BLS コース、または AHA PEARS コースの修了を確認できる書類が提出されていることで代える
ことも可とする。 
（２）小児救急 一般目標および診療能力 12 項目中、8 項目以上の自己評価が A または B であるこ
と。 
 
Ⅷ. 小児総合診療 一般目標・診療能力 
80%以上の項目の自己評価が B 以上であること。ただし、自己評価が N である項目を分母に含める。 
 
Ⅸ．手技、症候・疾患 
80%以上の項目の自己評価が B 以上であること。 
 
Ⅹ．疾患 
①知識として十分に理解すべき代表的疾患 
②治療もしくは初期対応まで修得すべき代表的疾患 
① ②いずれも自己評価 B 以上が 80%以上であること。
