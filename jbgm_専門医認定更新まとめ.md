# 総合診療専門医 認定・更新 情報まとめ

> 出典: [日本専門医機構 総合診療専門医検討委員会](https://jbgm.org/)
> 2026年3月時点で収集した情報

---

## 1. 専門医の新規認定

### 認定要件（5つすべて必要）

1. 有効な医師国家資格および臨床研修修了実績
2. 書類審査の通過（研修修了証明書、研修実績証明、達成度評価記録、共通講習修了実績、学術業績、医師免許証写し）
3. 総合診療専門医認定試験（筆記CBT + 面接）に合格
4. JMSB Online System+ 上で研修実績が承認済み
5. 専門医認定料を支払い済み

### 認定期間・認定料

| 項目 | 内容 |
|------|------|
| 認定期間 | **5年間**（審査後、遡って同年4月1日から） |
| 認定料 | **22,000円（税込）**（領域11,000円 + 機構11,000円） |

## 2. 第6回認定試験（2026年度）

| 項目 | 内容 |
|------|------|
| 試験形式 | CBT試験（全国会場）+ オンライン面接試験 |
| 出題数 | **全250問**（専門医共通10問、総合診療I/II 120問、内科80問、救急20問、小児科20問） |
| CBT試験日 | **2026年10月11日（日）** |
| 面接試験日 | **2026年10月12日（月祝）または 10月18日（日）** |
| 受験料 | **55,000円（税込）** |
| 出願期間 | **2026年4月15日（水）〜 5月12日（火）正午** |
| 合否発表 | **2027年1月下旬頃** |
| 認定開始日 | **2026年4月1日**（認定期間5年間） |

### 受験資格（5区分）

| 区分 | 対象 |
|------|------|
| (イ) | 2018年度以降に研修を開始し、2026年3月31日までに3年以上の研修を修了（見込み含む） |
| (ロ) | 特任指導医（専攻医を6か月以上指導した経験あり） |
| (ハ) | 特任指導医（指導歴6か月未満だが、学生・研修医等への指導経験あり） |
| (ニ) | 2017年度家庭医療専門研修開始者（家庭医療専門医試験合格または研修修了者） |
| (ホ) | 再受験者・既研修修了で未受験者 |

### 再受験ルール

- 原則としてCBT試験と面接試験は同年に受験・合格が必要
- 片方のみ合格の場合、次回は不合格科目のみ受験可能
  - 専攻医：研修修了から5年以内
  - 特任指導医：2030年度まで
  - 家庭医療専門医：2027年度まで

## 3. 専門医の更新

### 更新要件（4つすべて必要）

1. 更新書類審査の通過（下記の単位取得を含む）
2. 更新試験（IBT形式）に合格
3. JMSB Online System+ に登録あり
4. 専門医更新料・システム使用料・領域講習受講料の支払い

### 更新料

**27,500円（税込）**（機構11,000円 + 検討委員会16,500円）

### 必要単位数：合計50単位以上

| 項目 | 単位数 |
|------|--------|
| B. 診療実績の証明 | **10単位**（7つの資質能力 × 各2事例 = 14事例を記載し審査承認） |
| C. 共通講習 | **最小8単位、最大10単位** |
| D. 領域講習 | **最小20単位** |
| E. 学術業績・診療以外の活動実績 | **最大10単位** |
| 更新試験合格 | 必須（単位には含まず） |

### 共通講習の内訳

| 区分 | 科目 | 備考 |
|------|------|------|
| **必修A（全員必須）** | 医療安全、感染対策、医療倫理 | 5年間で各1単位以上 |
| **必修B（全員必須）** | 医療制度と法律、地域医療、医療福祉制度、医療経済、両立支援 | 各1単位以上 |
| **任意C** | 臨床研究・臨床試験、災害医療 など | 任意 |

- 「多様な地域における診療実績」が認定された場合、共通講習は最小3単位（必修Aのみ）に軽減され、必修Bが免除

### 領域講習の内訳（合計20単位以上）

| 種別 | 単位数 |
|------|--------|
| (1) 機構の総合診療講習会 | **最小10単位** |
| (2) 18基本領域学会 | 最大10単位 |
| (3) サブスペシャルティ領域学会 | 最大4単位 |
| (4) 日本医学会総会 | 最大4単位 |
| (5) 海外学会（WONCA, AAFP, STFM等） | 最大6単位 |
| (6) 指導医講習会 | **最小2単位**、最大6単位 |
| (7) 検討委員会認定のその他講習会 | 最大6単位 |
| (8) (7)の団体同士が共催する学術集会 | 最大15単位 |

### 学術業績の単位算定（E項目）

| 活動 | 単位 |
|------|------|
| 学術集会の筆頭/第2発表者、司会、座長 | 1回1単位 |
| 試験委員・面接委員・査読者 | 1年度1単位 |
| ピアレビュー論文の筆頭/第2著者 | 1回3単位 |
| ピアレビュー論文の第3以降著者 | 1回1単位 |
| 領域講習コンテンツ作成者・テキスト著者 | 1回1単位 |

### 勤務実態の要件

- 総合診療専門医としての活動時間が **週24時間以上** 必要
- 診療・教育・研究・行政が対象
- 24時間未満の場合は申告書を提出し審査

### 多様な地域における診療実績

- 更新1期目までに「医療資源の乏しい地域」で1年間の診療実績が望ましい
- 困難な場合は3回目の更新時（約15年間）までに達成

## 4. 更新試験（IBT）

### 第2回更新試験（2026年度）

| 項目 | 内容 |
|------|------|
| 試験形式 | **IBT形式**（自宅・職場等でのオンライン受験） |
| 出題数 | **全100問**（総合診療領域） |
| 試験時間 | **200分**（1問あたり上限3分、超過で自動遷移） |
| オープンブック | テキスト・書籍等の参照可（電話・チャット・撮影は不可） |
| 試験日 | **2026年10月11日（日）午前** |
| 受験料 | **22,000円（税込）** |
| 出願期間 | **2026年4月15日（水）〜 5月12日（火）正午** |
| 合否発表 | **2027年1月下旬頃** |

### 受験資格

| 区分 | 対象 |
|------|------|
| (イ) | 2022年度認定者（認定5年目、2027年3月満了） |
| (ロ) | 2023年度認定者（認定4年目、2028年3月満了） |
| (ハ) | 2021年度認定者で猶予/休止により2027年3月満了の者 |

### 受験回数

- 認定後の4年目および5年目に **最大2回** 受験可能
- 2回とも未受験/不合格の場合 → 猶予申請で更新年に1回のみ追加受験可

## 5. 移行措置

### 対象者と期限

| 対象 | 移行措置期間 |
|------|-------------|
| 特任指導医（専攻医指導6か月以上） | **2022年度〜2030年度** |
| 特任指導医（指導歴6か月未満/学生等への指導） | **2022年度〜2030年度** |
| 2017年度家庭医療専門研修開始者 | **2023年度〜2027年度** |

- 通常の専門医試験と同一の試験に合格する必要あり
- 移行措置で取得した専門医の更新は通常の更新基準と同一

## 6. 特任指導医・指導医の認定・更新

### 特任指導医

| 項目 | 内容 |
|------|------|
| 認定要件 | 卒後臨床経験7年間 |
| 認定日 | 8年目以降の4月1日 |
| 認定期間 | 5年間 |
| 講習会 | 認定の前年度に受講が必要 |
| 更新 | 認定から5年後の4月1日。更新前年度に講習会の再受講が必要 |
| 移行措置期間 | 2022〜2030年度 |

### 指導医

- 最速で初回の専門医更新時に指導医認定可能
- 指導医認定から5年目に専門医・指導医資格が同時更新

## 7. 更新困難な場合の救済措置

| 制度 | 内容 |
|------|------|
| **活動休止** | 初回最長2年、以降1年ごとに延長可能（上限5年）。休止期間を除く前後5年で更新基準を満たす。休止中は専門医を名乗れない |
| **更新猶予** | 1年間の猶予が可能（6年目に更新）。猶予期間中も専門医資格は維持 |
| **資格回復** | 失効後1年以内に更新基準を満たせば復活可能 |
| **資格取消** | 虚偽申請や不適切行為があった場合に取消、認定証返還が必要 |

## 11. 学修コンテンツ一覧（領域講習）

2026年2月時点で33以上のオンラインコンテンツが提供されている（各50〜57分、1単位、2,200円）。

### 主要テーマ

- **症候学**: 頭痛、けいれん、意識障害、浮腫、貧血、関節痛 等
- **感染症**: 呼吸器感染症、皮膚感染症、抗菌薬
- **循環器**: 胸痛、動悸、心筋梗塞、ACLS
- **代謝内分泌**: 糖尿病、脂質異常症、甲状腺、肥満症
- **消化器**: 腹痛、便秘、下痢
- **神経系**: 脳梗塞、てんかん
- **小児**: 発熱、中耳炎
- **老年医学**: フレイル、老年症候群、高齢者総合機能評価
- **その他**: 予防医療、患者中心の医療、プロフェッショナリズム、臨床倫理、研究・教育技法

### 指導医講習コンテンツ

| コード | 内容 | 単位 | 費用 |
|--------|------|------|------|
| GM6-1 | 総合診療専門研修概略 | 1単位 | 2,200円 |
| GM6-2 | 指導医・専攻医管理セット（ハラスメント・メンタルケア等） | 2単位 | 4,400円 |
| GM6-3 | 専攻医評価セット（360度評価・Mini-CEX・CbD等） | 2単位 | 4,400円 |
